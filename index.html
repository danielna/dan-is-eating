dan is eating

I kind of hate flickr's interface. I'd rather have an easier means of getting to the photos of food I'm uploading from my iPhone.

Using the flickr API and the jQuery isotope library for the pinterest-esque layout below. CSS is written in Sass with the help of Compass. Handlebars.js is used to render some JavaScript.

Hover an image to view the restaurant. Click an image to view the meal description, as well as a link to the photo on flickr to see the image in a higher resolution. Use the buttons at the top to filter between tags.

I mess with this page whenever I'm bored, so it will be sporadically updated. View the source on github.
