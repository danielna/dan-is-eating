dan is eating

I kind of hate flickr's interface. I'd rather have an easier means of getting to the photos of food I'm uploading from my iPhone.

Using the flickr API and the jQuery isotope library for the pinterest-esque layout below. CSS is written in Sass with the help of Compass, for no real reason other than I think they're cool.

Hover an image to view its name. Keep hovering for the description. Click it to open it on flickr in a new window, primarily to see it in a higher resolution. Use the tags to filter.

I mess with this page whenever I'm bored, so it will be sporadically updated. View the source on github.
